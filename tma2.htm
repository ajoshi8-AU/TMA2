Project 2: Learning Management System (LMS)

This LMS enables learners to register, log in, and access interactive lessons with embedded quizzes. All learning content is authored in a custom XML-based Educational Markup Language (EML), parsed and rendered server-side using PHP.

System Analysis & Design

  • Single-table database design (`lessons` table) with EML lesson data
  • All lessons are dynamically parsed and styled consistently
  • Quizzes are embedded in the lesson data and toggled with JavaScript
  • PHP parser extracts quizzes and formats them into interactive forms
  • JavaScript used to grade and display feedback without reloading the page

Educational Markup Language (EML)

The EML format was designed for non-technical authors. It includes the following tags:

  • <lesson>: root element
  • <title>, <content>, <code>: for instructional text
  • <quiz>, <question>, <text>, <option correct="true|false">: for multiple-choice assessments

All content is stored as EML in the database and parsed using parse_eml.php.

Database Design

  • Database name: lms_db
  • users: stores account credentials
  • lessons: stores title, course, and EML content

SQL setup script: part2/database.sql

Parser

The parse_eml.php script converts custom EML content into valid HTML, including quiz forms with radio buttons and JavaScript integration for grading. The quiz only becomes visible when the user clicks "Take Quiz".

Implementation

  • Users register and log in to access lessons from a dashboard
  • Lesson pages display parsed EML content
  • JavaScript toggles and grades quizzes dynamically
  • AJAX allows content display without full-page reloads
  • Responsive and consistent layout using styles.css
