Project 2: Learning Management System (LMS)

This LMS allows learners to register and log in to view lessons and take quizzes. Lessons and quizzes are marked up in a custom Educational Markup Language (EML), stored in a database, and parsed into HTML using PHP.

System Analysis & Design

  • User login/registration
  • Database stores lesson content and quizzes in EML
  • PHP parser translates EML to HTML
  • JavaScript and AJAX used for dynamic content updates

Educational Markup Language (EML)

The EML format was designed to allow subject matter experts (SMEs) to define lessons and quizzes. It includes tags like <lesson>, <content>, <quiz>, and <question>. The XML structure is documented in the report and stored in the database.

Database Design

The LMS uses a single MySQL table (lessons) with columns for ID, title, content (EML), and type (lesson/quiz). SQL scripts are provided in part2/database.sql.

Parser

A PHP script parses the custom EML into HTML on the server side before rendering the lesson or quiz for the user. This allows real-time delivery of learning material without manual HTML conversion.

Implementation

  • PHP handles parsing and session control
  • MySQL stores lessons and user data
  • JavaScript and AJAX are used to load tools and quizzes dynamically without page reload

User Guide

To access the LMS: Learning Management System

Learning Content

The LMS includes lessons on HTML5, CSS, and JavaScript. Each lesson includes embedded quizzes stored in XML and dynamically rendered.
